Reflective Room Manager is an easy-to-use, modular, and adaptive advanced room system for Unity and Mirror.

*** IMPORTANT ***
If mirror is not installed in the project, it will give an error. Make sure to install the mirror library into the project first!

Requirements:
    Unity 2019 / 2022 LTS
    Runtime .Net 4.x (Project Settings > Player > Other Settings)

Documentation:
    https://reflective-roommanager.gitbook.io/docs/
